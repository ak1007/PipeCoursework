Platform Independent Petri Net Editor 2
=======================================

http://pipe2.sourceforge.net

To use:

Set your JVM classpath to the bin/ directory and the default class to "RunGui". For example:

  on Linux:    java -cp ~/pipe/bin RunGui

  on Windows:  java -cp "c:\Program Files\pipe\bin" RunGui


Mac users can just double-click on bin/RunGui.class.


PIPE is open-source. If you wish to contribute then contact the admins listed on the PIPE SourceForge site.
